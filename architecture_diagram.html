System Architecture Overview

Sentinel2 is a comprehensive predictive maintenance system that combines traditional ML models with Google ADK integration for enhanced equipment intelligence and real-time monitoring.

Data Layer

  • Equipment sensor data (CSV)
  • Maintenance logs with timestamps
  • Incident reports and root causes
  • Predictive metrics storage
  • Real-time data ingestion

FastAPI Backend

  • RESTful API endpoints
  • Static file serving
  • Jinja2 template rendering
  • CORS and security middleware
  • Health check endpoints

ML Core Engine

  • Random Forest Classifier
  • Gradient Boosting Regressor
  • Isolation Forest (Anomaly Detection)
  • LSTM Networks (Optional)
  • Feature engineering pipeline

Google ADK Integration

  • Equipment Intelligence Agent
  • Google Search Agent
  • Real-time equipment research
  • Enhanced RUL predictions
  • Component-specific analysis
